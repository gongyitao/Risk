# 策略E+ 智能化系统 - 快速启动指南

## 🚀 5分钟快速上手

### 第一步：打开项目

**最简单的方式**：
1. 找到项目文件夹 `风控策略平台规划`
2. 双击打开 `index.html` 文件
3. 系统会自动在默认浏览器中打开

**推荐方式（使用本地服务器）**：
```bash
# 在项目目录下运行以下命令之一

# 如果你有Python 3
python -m http.server 8000

# 如果你有Node.js
npx http-server -p 8000

# 如果你有PHP
php -S localhost:8000
```

然后访问：http://localhost:8000

## 💡 演示技巧

### ✅ DO（推荐做法）

1. **提前准备**
   - 在演示前先自己操作一遍
   - 确认所有功能正常运行
   - 准备好演示话术

2. **流畅演示**
   - 使用快捷建议而不是手动输入（避免打字错误）
   - 等待动画完整播放（体现流畅性）
   - 边操作边解说（不要冷场）

3. **强调亮点**
   - 突出时间节省（4小时→30分钟）
   - 突出准确率（代码可用率85%+）
   - 突出业务价值（坏账率降低5%）

4. **互动引导**
   - 询问观众是否有类似痛点
   - 邀请观众提出需求场景
   - 收集反馈和建议

### ❌ DON'T（避免做法）

1. **不要纠结细节**
   - 这是原型，不是成品
   - 数据是模拟的，不是真实的
   - 重点在展示思路和交互

2. **不要过度承诺**
   - 说明这是演示版本
   - 实际开发需要时间
   - 某些功能还在规划中

3. **不要忽视问题**
   - 如果有bug，诚实说明
   - 如果功能缺失，记录需求
   - 如果性能不佳，说明优化计划

## 🔧 常见问题

### Q1: 页面打不开或样式错乱？
**A**: 
- 检查是否使用了较新版本的浏览器（推荐Chrome/Edge）
- 确认所有文件都在同一目录下
- 尝试使用本地服务器而不是直接打开文件

### Q2: 图表不显示？
**A**: 
- 检查网络连接（需要加载Chart.js库）
- 等待几秒让CDN资源加载完成
- 或下载Chart.js到本地引用

### Q3: 点击按钮没反应？
**A**: 
- 打开浏览器开发者工具（F12）查看控制台错误
- 确认JavaScript文件路径正确
- 刷新页面重试

### Q4: 想修改演示数据？
**A**: 
- 打开 `assets/js/workspace.js` 文件
- 找到 `mockData` 对象
- 修改其中的数据即可

### Q5: 想调整界面颜色？
**A**: 
- 打开 `assets/css/style.css` 文件
- 找到 `:root` 部分的CSS变量
- 修改颜色值即可全局生效
