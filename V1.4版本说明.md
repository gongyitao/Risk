# V1.4版本说明 - 双工作模式

## 🎊 重大更新

回答您的问题：**顶部四个模块现在支持两种使用方式！**

---

## 📊 您的问题解答

### Q1: 四个模块是什么作用？

**A**: 四个模块既可以**独立使用**，也可以按**工作流**顺序使用：

#### 模块功能

| 模块 | 独立使用 | 工作流位置 |
|-----|---------|-----------|
| 📊 数据助手 | 快速查询数据 | 步骤1：取数清洗 |
| 🔍 策略挖掘 | 特征分析 | 步骤2：分析挖掘 |
| 📄 报告生成 | 生成报告 | 步骤3：出报告 |
| 📚 知识库 | 合规查询 | 随时可用 |

---

### Q2: 能独立运行吗？还是必须按工作流？

**A**: **两种都支持！** 我们特意设计了双模式系统：

#### 🆓 模式1：自由模式（独立使用）

**特点**：
- ✅ 可以直接跳到任何模块
- ✅ 不需要按顺序
- ✅ 灵活快速

**使用场景**：
```
场景1：只想快速查询数据
→ 直接用"数据助手"

场景2：只想生成报告
→ 直接用"报告生成"

场景3：只想查合规
→ 直接用"知识库"
```

---

#### 🔁 模式2：工作流模式（按流程）

**特点**：
- ✅ 系统引导按步骤操作
- ✅ 步骤1→2→3完整流程
- ✅ 自动提示下一步

**工作流程**：
```
步骤1：数据助手
  ↓ 取数 → 清洗 → 准备完成 ✓
  
步骤2：策略挖掘
  ↓ 特征分析 → 分箱 → 推荐策略 ✓
  
步骤3：报告生成
  ↓ 回测 → 换手分析 → 报告完成 ✓
  
随时：知识库（合规检查）
```

---

## 🎯 如何切换模式？

### 切换按钮位置
在页面顶部，四个模块右侧，有一个**"工作流"按钮**。

### 操作方法
```
点击按钮 → 立即切换模式

自由模式 ⇄ 工作流模式
```

### 视觉区别

**自由模式**：
- 按钮显示"工作流"
- 模块无步骤编号
- 随意点击任意模块

**工作流模式**：
- 按钮显示"自由模式"（高亮）
- 模块显示"步骤1/2/3"
- 完成步骤有✓标记
- 系统自动引导下一步

## 💡 使用建议

### 什么时候用自由模式？

✅ **推荐场景**：
- 你很熟悉系统
- 只需要某个特定功能
- 时间紧急
- 需要频繁切换功能

**例子**：
```
- 只想查询昨天的逾期数据 → 用自由模式
- 只想看某个字段是否合规 → 用自由模式
- 数据已有，只需生成报告 → 用自由模式
```

---

### 什么时候用工作流模式？

✅ **推荐场景**：
- 你是新手
- 需要完整的策略分析
- 需要向他人演示流程
- 想确保不遗漏步骤

**例子**：
```
- 新人培训 → 用工作流模式
- 开发新策略（从零开始）→ 用工作流模式
- 标准化分析流程 → 用工作流模式
```

---

## 📋 实际案例对比

### 案例1：快速查询

**场景**：只想看看昨天的逾期用户数量

**自由模式（推荐）**：
```
1. 点击"数据助手"
2. 输入查询需求
3. 查看结果
完成！耗时：1分钟
```

**工作流模式**：
```
1. 启用工作流模式
2. 步骤1：数据助手查询
3. 系统提示"是否继续步骤2"
4. （实际不需要后续步骤）
完成！耗时：2-3分钟
```

**结论**：自由模式更快

---

### 案例2：新策略开发

**场景**：从零开发一个年轻用户风控策略

**自由模式**：
```
1. 数据助手 - 查询数据
2. （需要记住下一步做什么）
3. 策略挖掘 - 分析特征
4. （需要记住下一步做什么）
5. 报告生成 - 生成报告
完成！但可能遗漏某些步骤
```

**工作流模式（推荐）**：
```
1. 启用工作流模式
2. 步骤1：数据助手 ✓
3. 系统提示"继续步骤2" → 点击
4. 步骤2：策略挖掘 ✓
5. 系统提示"继续步骤3" → 点击
6. 步骤3：报告生成 ✓
7. 系统提示"流程完成"
完成！确保不遗漏，有引导
```

**结论**：工作流模式更完整

## 🎯 设计理念

### 为什么设计两种模式？

我们在调研时发现用户需求是多样的：

**痛点1**：新手不知道从哪开始
- 解决方案：工作流模式引导

**痛点2**：老手觉得步骤繁琐
- 解决方案：自由模式随意跳转

**痛点3**：不同任务需求不同
- 解决方案：一键切换，灵活应对

### 核心设计原则

1. **松耦合**：模块可以独立运行
2. **可组合**：模块也可以按流程组合
3. **用户可控**：用户自己选择模式
4. **无障碍切换**：随时可以切换模式

## ❓ 常见问题

### Q: 是否必须按步骤来？
**A**: 不是！即使在工作流模式下，也可以随时跳到任意模块。工作流只是提供引导，不是强制限制。

### Q: 步骤编号会一直显示吗？
**A**: 只有在工作流模式下才显示。切换到自由模式后编号会隐藏。

### Q: 完成标记✓会保存吗？
**A**: 当前会话中会保存。刷新页面后会重置。

### Q: 可以自定义工作流吗？
**A**: 当前版本是固定的1→2→3流程。未来版本将支持自定义工作流。

### Q: 知识库为什么不在工作流里？
**A**: 因为知识库（合规查询）是辅助功能，可以在任何时候使用，不影响主流程。
