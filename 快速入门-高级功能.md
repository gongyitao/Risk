# 🚀 高级功能快速入门

## 一、智能反问确认功能

### ⚡ 最快方式
**点击快速建议按钮** → **"💡 体验反问确认（示例）"**

### 📝 或输入以下内容
```
查一下那些坏客户
分析一下年轻用户
看看优质客户
```

### ✨ 效果
系统会自动弹出反问确认对话框，提供快速选择建议。

---

## 二、交互式拖拽分箱功能

### ⚡ 最快方式
**点击快速建议按钮** → **"🎯 交互式分箱分析"**

### 📝 或输入以下内容
```
挖掘特征
分箱分析
帮我挖掘高风险客群特征
```

### ✨ 效果
右侧会显示交互式分箱图表，您可以：
- 🖱️ **拖拽切分点**：鼠标拖拽图表上的圆点手柄
- 📊 **实时查看统计**：IV值、坏账率、单调性等
- 🔄 **切换特征**：年龄、收入、入网时长、借款金额
- ✅ **应用方案**：点击"应用分箱"保存方案

---

## 🎯 推荐操作流程

### 体验反问确认
1. 打开 `workspace.html`
2. 点击 **"💡 体验反问确认（示例）"**
3. 查看系统反问对话框
4. 点击快速建议或输入详细描述

### 体验拖拽分箱
1. 打开 `workspace.html`
2. 点击 **"🎯 交互式分箱分析"**
3. 等待图表加载（1-2秒）
4. 鼠标移到切分点手柄上，拖拽调整
5. 观察实时更新的统计数据

---

## 💡 提示

- ✅ 两个功能都可以通过**快速建议按钮**一键进入
- ✅ 也可以直接在对话框中输入相关关键词
- ✅ 反问确认会自动触发，无需特殊操作
- ✅ 分箱功能需要等待图表加载完成
